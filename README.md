**Project Title: To-Do List Application**


**Description:** This is a Python-based To-Do List application with multiple task management. It allows users to add one or more tasks (comma-separated), view all tasks with numbering, and remove tasks by their index. The program runs in a simple command-line interface and demonstrates Python fundamentals such as lists, loops, error handling, and user input. It is lightweight, beginner-friendly, and helps users organize their daily activities efficiently.


**Tech Stack:-** Python 3.x


**Installation Instructions:-**

Clone the repository: git clone https://github.com/Jayaram4210/To-Do-List-Application.git cd To-Do-List-Application

Ensure Python 3.x is installed on your system.

Run the project: python To-Do-List-Application.py



**Features:-**

Add one or more tasks (comma-separated)

View all tasks with numbering

Remove tasks by index

Error handling for invalid inputs

Lightweight and beginner-friendly command-line interface



**Troubleshooting:-**

Ensure Python is correctly installed and added to PATH

Run the script from the project directory

Enter valid task numbers when removing tasks

Use comma-separated values when adding multiple tasks
